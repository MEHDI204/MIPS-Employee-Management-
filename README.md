# Système de Gestion d'Employés - MIPS

Mini-projet d'assembleur MIPS pour la gestion simplifiée d'une entreprise.

## Fonctionnalités
- Affichage de la liste des employés
- Recherche par identifiant
- Calcul salaire Min/Max
- Calcul salaire moyen
- Statistiques salariales

## Utilisation
1. Ouvrir avec MARS MIPS Simulator
2. Assembler (F3)
3. Exécuter (F5)
